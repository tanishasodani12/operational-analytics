# Operational Analytics & User Engagement Study (SQL)

This project focuses on analyzing user engagement trends, job review activity, device usage patterns, and workflow efficiency using SQL queries. The goal was to identify performance bottlenecks and provide insights that can support decision-making in improving user engagement and operational throughput.
